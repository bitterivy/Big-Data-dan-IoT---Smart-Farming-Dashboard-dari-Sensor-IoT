# Smart Farming Dashboard

Project ini merupakan analisis data **Smart Agriculture** menggunakan dataset sensor pertanian untuk memonitor kondisi lingkungan seperti suhu, kelembaban udara, dan kelembaban tanah.

## Dataset

Dataset yang digunakan adalah **cropdata_updated.csv** yang disimpan pada folder:

data/raw/

## Data Processing

Proses yang dilakukan:

* Data cleaning
* Analisis statistik data sensor
* Visualisasi data

## Dashboard

Dashboard dibuat menggunakan **Power BI** untuk menampilkan visualisasi kondisi sensor pertanian.

File dashboard:

dashboard/smart_farming_dashboard.pbix

## Outputs

Hasil analisis disimpan pada folder **outputs**:

* cleaned_data.csv
* analysis_report.pdf
* dashboard_screenshot.png
